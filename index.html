What is AGRep?

Accessible Graphics Repository (AGRep) is an online repository that consists of a corpus of documents identified in a systematic literature review focused on touch-based accessible graphics. As a tool AGRep was designed to support the authors, as well as the wider touch-based graphics research community, in exploring the corpus of identified resources. AGRep is envisioned as a living resource capturing what has been done in the field of accessible graphics technology, and users can update the corpus with newly published resources in the future. AGRep has been customised to improve its level of accessibility, with efforts ongoing.

How to Use and Navigate AGRep

The tool is split into two main sections: category filtering and associated resources. Filters are presented in groups - e.g. Participation, Contributions, Source etc. Groups have multiple filters - e.g. Blind Participants, Low Vision Participants, Blindfolded Participants - each represented by an icon which is accessible by both keyboard and mouse. AGRep filter icons by default are loaded in an “active” state. Think of each filter icon as a light switch which has been turned on, causing all resources in the corpus to load by default. To filter the corpus, select and deactivate the filter icons that you would like to hide (“flip the light switch”) - e.g. if you would like to view all resources that involved Blind Participants, deselect all other Participation filter icons.

Listed resources are updated dynamically according to user filtering. Resources are accessible by both keyboard and mouse and can be tabbed into. When a resource has been opened information relevant to that particular resource will be accessible, including resource metadata such as authors, venue and how it was categorised by our research team.
